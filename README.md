# 🍽️ Swad Bharat (Swad-Bharat)
**Explore the soul of India through its flavors.** Swad Bharat is a comprehensive, interactive recipe platform designed to showcase the culinary heritage of every Indian state. From the street foods of Mumbai to the royal kitchens of Rajasthan, we bring traditional recipes to your fingertips.

## 🚀 Core Features
* **Interactive Map/State Gallery:** Browse high-quality visuals of every Indian state to discover their unique food culture.
* **Extensive Recipe Database:** Each state features 12-15 famous dishes with detailed, step-by-step cooking instructions.
* **User Dashboard & Accounts:** Users can manage their profiles and track their contributions.
* **Favorites (Wishlist):** A dedicated section to save and organize recipes you love.
* **Recipe Contribution:** An "Add Your Own Recipe" feature that allows users to share their family secrets with the community.
* **Smart Search:** Find any dish or ingredient instantly using the integrated search bar.
* **Responsive About & Contact:** Clear communication channels and a professional team overview.

## 🛠️ Project Structure
* **State-Specific Modules:** Organized navigation from State → Recipe List → Cooking Guide.
* **Dynamic UI:** Built with clean HTML5 and CSS3 for a smooth visual experience.
* **Functionality:** JavaScript-driven search and filtering systems.
